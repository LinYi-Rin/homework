## 选题简介
本次视频围绕**具身智能（Embodied AI）**展开，用通俗语言讲解：
1. 什么是具身智能
2. 它和传统 AI 的区别
3. 核心技术：感知、决策、执行
4. 典型应用场景：家庭服务机器人、工业机械臂、自动驾驶、医疗辅助机器人等
5. 未来发展趋势

视频目标是让没有专业背景的观众也能快速理解：AI 如何从“只会思考”走向“能动手、能行动”的物理世界智能体。
